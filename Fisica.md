# Capacidade Térmica 
É  a quantidade de calor necessaria para se elevar em 1°C a temperatura de um corpo/um objeto
```
C = Q/△T
Q -> Quantidade de calor (Cal)
△T -> Variação de temperatura (°C)
C -> Capacidade térmica (Cal/°C)
```
